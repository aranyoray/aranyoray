# Aranyo Ray - Portfolio Website

A modern, minimalist portfolio website inspired by clean design principles.

## Features

- ✨ Smooth scroll animations
- 🎨 Interactive project cards with 3D hover effects
- 📱 Fully responsive design
- 🎯 Custom cursor interactions
- ⚡ Optimized performance
- 🎨 Beautiful gradient backgrounds
- 🔄 Staggered content loading

## Setup

1. Replace `arlogo.avif` with your actual logo file
2. Update the email address in the "Get in touch" button
3. Update social media links in the footer
4. Customize project cards with your actual projects
5. Update timeline with your experiences

## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --color-accent: #2d5a27;  /* Your brand color */
    --color-bg: #fdfcfa;      /* Background */
}
```

### Content
- Hero section: Edit `index.html` lines 25-35
- Projects: Edit project cards starting at line 40
- Timeline: Edit timeline items starting at line 150
- About section: Edit lines 185-200

## File Structure

```
├── index.html       # Main HTML structure
├── styles.css       # All styling
├── script.js        # Interactions and animations
├── arlogo.avif     # Logo file (replace with yours)
```
